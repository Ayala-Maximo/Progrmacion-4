Como dar seguimiento a archivos con git

Por : Maximo Ayala

Para hacer seguimiento a archivos con git nesecitaremos primero nesecitaremos instalarlo Video Tutorial para su intalacion. Tras instalarlo podremos hacer seguimientos, ramas y retoceder cambios de cualquier tipo de archivo ¿y para que no sirve esto? bueno hay multiples razones por que nos pueda llegar aser utiles tales como :

  • Podemos Volver a versiones antiguas del archivo
  • Podemos hacer varias rama distinta del archivo para comparar
  • Podemos hacer un registro de los cambios

Y ¿Como hacemos para hacer seguimientos a nuestros archivos? simple siguiendo los siguientes pasos

  1. Crear un repositorio

    El primer paso sera crear un repositorio, que se hace selecionando con clic iziquierdo en una carpeta que servira como recipiente, con GIT BASH

    Luego nos saldra la terminal de git

    Luego tendremos que colocar el siguiente comando : git init

  2. Agregar un archivo

    Podemos guardar cualquier tipo de archivo pero para hacelo facil usaremos un .txt, primero lo creamos y le daremos nombre ("Hola Mundo")

    depues en el GIT Bash escribimos git add . esto hara que guarde todos los archivos en el repositorio
